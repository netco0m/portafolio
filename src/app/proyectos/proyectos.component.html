Hidesk - sistema de contratos y facturación

Hidesk es un sistema integral de administración empresarial diseñado específicamente para gestionar todas las operaciones de Himce S.A.. Este proyecto nació de la necesidad de sustituir el manejo manual con Excel, proporcionando una solución robusta que cubre todos los aspectos de la empresa. Actualmente, Hidesk se utiliza para administrar múltiples sucursales y departamentos en todo el país.

Entre sus funcionalidades principales, Hidesk permite la creación y gestión de contratos, generación de informes de facturación, y un control centralizado para servicios ofrecidos por Himce S.A., como el arriendo de baños químicos, servicios de seguridad, servicios especiales, y servicios de aseo.

La suite de opciones del sistema incluye:

  • Administración: Descarga de documentos, gestión de clientes, sucursales, obras, artículos, servicios de contrato, usuarios, perfiles, correos, e instalaciones (como baños químicos).
  • MIG Seguridad: Modificación de contratos, facturación general y por sucursal, y estado de contratos en este departamento.
  • Servicios Especiales: Gestión de contratos, facturación general y por sucursal, y estado de contratos específicos.
  • Aseo: Creación y edición de contratos, facturación, y seguimiento de contratos.
  • Autorización de Contratos: Validación y autorización de contratos en todos los departamentos (baños químicos, seguridad, servicios especiales, y aseo).
  • Supervisión: Monitoreo de operaciones.
  • Gestión de Vehículos: Control y asignación de flota.
  • Tickets: Seguimiento y resolución de solicitudes internas.

Este sistema ha permitido a Himce S.A. optimizar procesos, garantizar un mejor control operativo, y centralizar todas las operaciones de manera eficiente y segura.

E-commerce

Este proyecto fue desarrollado como una solución personalizada para la venta de productos de belleza. Construido utilizando Laravel y Vue.js, cuenta con integración certificada con Transbank para aceptar todos los medios de pago digitales. Fue desplegado en servidores AWS Linux y gestionado mediante Git en GitHub. Entre sus funcionalidades se destacan:

  • Pantalla de inicio con productos categorizados por marcas y promociones.
  • Carrito de compras intuitivo y fácil de usar.
  • Sistema de intranet con roles y permisos para gestionar productos, usuarios y ventas.
  • Pago seguro mediante Transbank y certificados SSL.
